# Source Code Directory

## Structure
```
src/
├── data_collection/     # Scripts for gathering data
├── data_processing/     # Data cleaning and transformation
├── machine_learning/    # ML models and classifiers  
├── statistical_analysis/# Statistical modeling
└── visualization/       # Plotting and reporting
```

## Adding Your Code
1. Copy your Python scripts to the appropriate subdirectories
2. Update import paths as needed
3. Add requirements.txt for dependencies
4. Document any new modules in this README
